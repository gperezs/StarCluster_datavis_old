NGC 1566

Scroll the target image between filtered (backlight removed) and non-filtered:

In order to visualize the objects, pixel values inside [mean-std,mean+std] of each band of each target were used. Then, the image was filtered to remove the background light. The RGB image was formed by combining bands like this:

  • Blue = (21.63*F275W + 8.63*F336W) / 2
  • Green = 4.52*F438W
  • Red = (1.23*F555W + F814W) / 2

The mosaics are created by cropping 22x22x3 boxes centered on each object coordinates (given in the .tab annotations).
